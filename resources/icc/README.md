# ICC profiles

`sRGB2014.icc` — the sRGB v2 profile published by the International Color
Consortium (<https://www.color.org/srgbprofiles.xalter>). Distributed here
unmodified, as permitted by the ICC profile license terms ("this profile may
be used, copied and distributed provided it is not altered").

Used as the `/OutputIntents` destination profile for the PDF/A and PDF/X
conformance reference documents (`scripts/conformance/`) and available to
users as a ready-made profile for `PdfAConfig` / `PdfXConfig`.
