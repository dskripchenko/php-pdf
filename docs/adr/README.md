# Architecture Decision Records — php-pdf

Формат: каждое решение — отдельный markdown с структурой
`Context → Decision → Consequences` (Michael Nygard style). Номера
сквозные, не переиспользуются. Префикс `PDF-` отличает ADR этой
библиотеки от ADR-NNN в parent-проектах (printable, laravel-admin).
